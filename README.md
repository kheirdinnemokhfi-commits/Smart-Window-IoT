# 🌐 Smart Window IoT System

An intelligent IoT-based smart window and blinds automation system using ESP8266.

---

## 🚀 Features

- 🌧 Automatic rain detection
- ☀️ Light-based blinds control using LDR
- 🌡 Temperature & humidity monitoring
- 📱 Remote web dashboard
- ⚡ Real-time monitoring
- 🕒 Scheduled automation modes

---

## 🛠 Hardware Components

| Component | Description |
|---|---|
| ESP8266 WeMos D1 | Main controller |
| DHT11 | Temperature & humidity sensor |
| LDR Sensor | Light detection |
| Rain Sensor | Rain detection |
| Servo Motors | Window & blinds movement |
